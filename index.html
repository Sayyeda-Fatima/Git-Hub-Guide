An easy guide to Git

Introduction

This is a beginer friendly guide to Git.

What is Git?

Git is a Distributed Version Control System. In addition to being distributed, Git has been designed with performance, security and flexibility in mind.
In this tutorial we will see some important Git commands and concepts that every beginer should know.

Let's begin then: First of all you will need to download and install Git on your computer. Once you have installed git now you can create local repisitories and start your version control.

Repository:

The first and most common concept of Git is repositories. All the file versions are saved in a repository. In simple words it is a folder that stores all the version and history of changes in our code.

Creating a Git Repository

To create a repository, you can press right click on mouse, press new from the menu and create a folder and open that folder in Command line terminal. Then we use command :

git init


After creating the folder open it and again press right click on mouse. Choose "Open in terminal" from menu. It will open a command line prompt in your folder. Now you can write " git init " command and initialize a git repository in that folder.





Now we have succesfully initialized a git repository in our folder. When we intialize a git repository in a folder a " .git " folder is created in our original folder and that .git folder is our repository but is hidden mostly.
