An easy guide to Git

Introduction

This is a beginer friendly guide to Git.

What is Git?

Git is a Distributed Version Control System. In addition to being distributed, Git has been designed with performance, security and flexibility in mind.
In this tutorial we will see some important Git commands and concepts that every beginer should know.

Let's begin then:

Repository:

The first and most common concept of Git is repositories. All the file versions are saved in a repository.

To create a repository, you can create a folder and open that folder in Command line terminal. Then we use command :

git init
